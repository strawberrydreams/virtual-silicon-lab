# Demo / Export QA

Capture `virtual-silicon-lab-demo.gif` here if a visual walkthrough is needed.

Recommended v2 flow:

1. Open `/` in desktop Chrome.
2. Switch `laboratory`, `anime`, and `space` page themes.
3. Open `/dashboard`.
4. Remix the first three v2 hero sets: `AURORA M5`, `PANTHER SCALE`, `N1 GREEN HORIZON`.
5. In each editor, select the matching poster format:
   - `AURORA M5`: `Press Hero`
   - `PANTHER SCALE`: `Architecture Slide`
   - `N1 GREEN HORIZON`: `Press Hero`
6. Download one die PNG and one poster PNG.
7. Confirm poster dimensions are `3200x1800` and no editor controls appear in the image.
8. Create one `Random Chip` from the dashboard and confirm it opens as an editable local project.
