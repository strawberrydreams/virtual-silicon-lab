# Hero Chip Compositions

One primary (A, built and reviewed first in M3) plus two alternates (B/C) that double as the
visual regression check that themes are genuinely distinct, not recolors. Block types come from the
real/fantasy palettes; a fantasy block anchors each composition.

## A — "AURORA C-1 — Consciousness Processor" (`keynote` + neon accent)

The reviewable first hero: keynote restraint, centered composition, one soft cyan→violet bloom.

- Die `square` on a deep-graphite radial spotlight; brushed space-gray; one accent gradient
  `#22d3ee`→`#a855f7`; exactly one soft bloom behind the central block.
- Blocks: `ConsciousnessProcessor` (fantasy hero, accent + bloom), `CPU`/`GPU` (top), `PLL`/`DAC`
  (bottom), `QuantumMemory` band (repeated-cell texture).
- Decorations: dotted perimeter bond-pad ring; one quiet name label; no neon lines.
- Fake spec: AURORA / C-1 / 3rd-gen / `0.5nm 영혼각인 (soul-etched)` / 88 cores / `∞ TB/s` /
  features Dream Coherence Engine, Lucid Cache, Empathy Co-processor.
- Review focus: soft (not hard-ring) bloom, real brushed-metal gradient depth, fine memory texture.

## B — "NEON DISTRICT N-9" (`neon`, hexagon die)

Full cyberpunk to stress `neon`: hexagon die, additive neon routing, HUD framing. Two accents only
(cyan `#22d3ee` + magenta `#ff2bd6`). `EmotionEngine` + `RealityDistortionUnit` as glowing anchors;
`neonLine` routing, `warningMark`, hex-vertex ticks. Spec: NEXUS / N-9 / 256 cores / 12.8 PB/s.

## C — "FIELD UNIT M-7" (`military`, rect die)

Counter-programs A/B to prove range: matte, heavy, almost no glow. Wide rect die, brushed
gunmetal/olive; `TimeCore` (fantasy) as the central sealed unit; hazard stripes, warning triangles,
stencil serials, rivets. Spec: AEGIS / M-7 / 32 cores / 2.4 TB/s.
